---

import ApikeyWarning from "/snippets/apikey-warning.mdx";

<ApikeyWarning />

Marks all messages of one subscriber as `read`, `unread`, `seen`, `unseen`. Optionally, single `feedIdentifier` or array of `feedIdentifiers` can be passed to mark messages of particular feeds.
