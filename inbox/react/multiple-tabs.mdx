Novu allows to create a multi-tab experience for Inbox, in a way you can fetch the notifications feed using `tags`.

### Defining tags

Define `tags` prop to show multiple tabs in the Inbox.
Each tag needs to have a `title` and `value` property. The `value` property can be a string or an array of strings.

```tsx
import { Inbox } from '@novu/react';

const tabs = [
  {
    title: 'All Notifications',
    value: [],
  },
  {
    title: 'Newsletter',
    value: ['newsletter'],
  },
  {
    title: 'React',
    value: ['react'],
  },
];

function Novu() {
  return (
    <Inbox
      applicationIdentifier="YOUR_APPLICATION_IDENTIFIER"
      subscriberId="YOUR_SUBSCRIBER_ID"
      tabs={tabs}
    />
  );
}
```
